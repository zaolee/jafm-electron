1 프로그램 소개

자금일보 관리 시스템은 소규모 기업의 일별 자금 입출금을 간편하게 관리하는 로컬 설치형 프로그램입니다. 인터넷 연결 없이 PC에서 단독으로 실행되며, 모든 데이터는 내 컴퓨터에만 저장됩니다.

📋 자금 입력

날짜·계좌·금액·거래처를 입력하여 입출금 내역을 등록하고 조회합니다.

📊 보고서

선택한 기간의 계좌별 자금 흐름을 보고서 형태로 확인하고, 인쇄 또는 Word 파일로 저장합니다.

💾 내보내기

엑셀 파일로 내보내거나, 은행 거래내역 파일을 자동으로 불러옵니다. JSON 백업·복원도 이곳에서 합니다.

⚙️ 설정

계좌 이름·번호·기준잔액을 관리하고, 보고서 결재란 항목 이름을 설정합니다.

주요 특징

  • 계좌 추가·삭제·수정 가능 — 회사 상황에 맞게 자유롭게 구성
  • 원화(₩) 및 외화($) 계좌 동시 관리
  • 마이너스 잔액 시 ⚠️ 경고 표시 및 빨간 글씨로 시각적 구분
  • 기업은행·농협·우리은행·국민은행·신한은행 거래내역 파일 자동 인식
  • 엑셀(.xls) · Word(.doc) 보고서 내보내기
  • JSON 백업으로 계좌설정·잔액·결재란·거래내역 전체 보관 및 복원
2 처음 시작하기 (초기 설정)

프로그램을 처음 설치한 후 아래 순서로 설정하면 바로 사용할 수 있습니다.

  • 1
    상단 탭에서 ⚙️ 설정을 클릭합니다.
  • 2
    계좌 설정 화면에서 계좌 이름·번호를 확인하고, 각 계좌의 기준 잔액에 현재 실제 잔액을 입력합니다.
    예) 기업은행 주계좌에 5,000만원이 있다면 50000000 입력 (콤마 없이 숫자만)
    기준 잔액 옆 현재잔액은 기준잔액 + 입금 − 출금으로 자동 계산되어 표시됩니다.
  • 3
    계좌 목록이 회사 상황과 다르면 ➕ 계좌 추가 버튼으로 추가하거나 🗑️ 버튼으로 삭제합니다.
  • 4
    ✅ 저장 버튼을 클릭합니다.
💡 기준잔액은 이후 입출금 내역의 계산 기준이 됩니다. 잘못 입력했다면 ⚙️ 설정 탭에서 언제든지 수정할 수 있습니다.
3 자금 입력 — 입출금 내역 등록

입출금 내역 입력

  • 1
    상단 탭에서 📋 자금 입력을 클릭합니다.
  • 2
    입출금 내역 입력 카드에서 아래 항목을 입력합니다.
    항목설명필수
    날짜거래 발생일필수
    계좌 구분거래가 일어난 계좌 선택 (설정에서 추가한 계좌 포함)필수
    입출금 구분💙 입금 또는 ❤️ 출금 선택필수
    금액숫자만 입력 (콤마 자동 표시)필수
    거래처 / 내용거래처명 또는 거래 내용필수
    비고추가 메모 (선택)선택
  • 3
    ✅ 저장 버튼을 클릭합니다. 저장 즉시 하단 내역 목록에 반영됩니다.

화면 상단 요약 정보

  • 💼 가용자금 합계 — 모든 원화·현금 계좌의 현재 잔액 합계 (외화 USD는 별도 표시)
  • 계좌 카드 — 각 계좌별 현재 잔액을 한눈에 확인. 잔액 마이너스 시 빨간 테두리 + ⚠️ 경고

내역 조회 및 필터

📋 입출금 내역 목록 카드에서 기간·구분·계좌 필터를 설정하면 원하는 내역만 볼 수 있습니다. 전체보기 버튼을 클릭하면 필터가 해제됩니다. 현재 보이는 목록만 엑셀로 내보내려면 📥 현재 목록 엑셀 버튼을 사용하세요.

내역 삭제

삭제하려는 내역의 오른쪽 삭제 버튼을 클릭합니다. 삭제 후에는 복구되지 않으니 주의하세요.

⚠️ 삭제된 내역은 복구할 수 없습니다. 중요한 데이터는 미리 JSON 백업을 해두세요.
4 보고서 — 주간 자금 현황 조회
  • 1
    상단 탭에서 📊 보고서를 클릭합니다.
  • 2
    보고 기간 시작일과 종료일을 선택합니다. (주 단위 조회 권장, 기본값: 이번 주)
  • 3
    보고서 양식을 선택합니다.
    • 공식 보고서 — 계좌별 잔액 현황 + 입·출금 통합 내역
    • 추가 양식 — 계좌별 잔액 현황 + 입금 내역 / 출금 내역 분리
  • 4
    🔍 보고서 생성 버튼을 클릭합니다.
  • 5
    필요에 따라 버튼을 선택합니다:
    • 🖨️ 인쇄 미리보기 — 인쇄 다이얼로그 열기 (프린터 출력 또는 PDF 저장)
    • 📄 Word 저장 — Word 호환 .doc 파일로 저장
💡 보고서에는 결재란(서명 공간)이 포함됩니다. 결재란 항목 이름(담당·이사 등)은 ⚙️ 설정 탭 → 결재란 설정에서 변경할 수 있습니다.
마이너스 금액은 ▼ 빨간색으로 표시됩니다.
5 내보내기 — 엑셀 · Word 저장 & 은행 파일 가져오기

💾 엑셀 파일 다운로드

  • 1
    상단 탭에서 💾 내보내기를 클릭합니다.
  • 2
    기간과 계좌를 선택합니다. (계좌 미선택 시 전체 계좌)
  • 3
    📥 엑셀 다운로드 — 선택 기간의 자금일보 양식 엑셀(.xls) 저장
    📦 전체 데이터 다운로드 — 모든 기간의 데이터를 한 번에 저장
    엑셀 파일은 입출금내역 시트 + 계좌잔액현황 시트 두 장으로 구성됩니다. 헤더는 파란색으로 표시됩니다.
📋 자금 입력 탭에서 필터를 적용한 후 📥 현재 목록 엑셀 버튼을 누르면 현재 보이는 목록만 엑셀로 저장할 수 있습니다.

📄 Word 보고서 저장

보고서 탭에서 기간과 양식을 선택 후 보고서 생성을 클릭한 다음 📄 Word 저장 버튼을 누릅니다. Word(.doc) 파일은 Microsoft Word에서 열 수 있으며, 표 서식이 그대로 유지됩니다.

🏦 은행 거래내역 가져오기

은행 앱/인터넷뱅킹에서 내려받은 거래내역 파일(.xlsx, .xls, .csv)을 자동으로 불러옵니다.

은행내보내기 경로
기업은행거래내역 조회 → 엑셀 저장
농협은행거래내역 조회 → 내보내기 → 엑셀
우리은행거래내역 → 파일저장 (Excel)
국민은행거래내역 조회 → 엑셀 다운로드
신한은행거래내역 조회 → 내보내기
기타자동 감지 선택 후 파일 업로드
  • 1
    은행 선택 — 파일을 받은 은행을 선택합니다. (잘 모르면 자동 감지 선택)
  • 2
    계좌 구분 — 가져올 내역을 어느 계좌로 등록할지 선택합니다.
  • 3
    📂 파일 선택 버튼을 클릭하여 파일을 선택합니다.
  • 4
    미리보기 화면에서 가져올 항목을 확인합니다. 불필요한 항목은 포함 체크를 해제합니다.
  • 5
    ✅ 선택 항목 가져오기를 클릭하면 자금 입력 탭에 자동 등록됩니다. 중복 내역은 자동으로 제외됩니다.
📋 은행별 엑셀 컬럼 구조 보기 버튼을 클릭하면 각 은행 파일의 컬럼 구조와 내보내기 방법을 자세히 확인할 수 있습니다.
6 계좌 설정 & 결재란 설정

⚙️ 계좌 설정

⚙️ 설정 탭에서 계좌 이름·번호·기준잔액을 자유롭게 편집할 수 있습니다.

항목설명
계좌 이름화면과 보고서에 표시되는 이름. 자유롭게 수정 가능
계좌번호참고용. 보고서에 표시됨. 없으면 빈칸
종류원화(₩) / 외화($) / 현금 중 선택
기준 잔액 ⓘ거래 입력 전 실제 잔액. 이후 입출금이 여기에 더해지거나 빠짐
현재잔액 ⓘ기준잔액 + 모든 입금 − 모든 출금. 자동 계산 (수정 불가)
  • ➕ 계좌 추가 — 새 계좌를 추가합니다. 추가 후 반드시 ✅ 저장 버튼을 눌러야 반영됩니다.
  • 🗑️ 삭제 — 거래내역이 없는 계좌만 삭제 가능합니다. 삭제 후 ✅ 저장을 눌러야 반영됩니다.
  • 🔄 되돌리기 — 저장하지 않은 변경사항을 원래대로 복원합니다.
⚠️ 현재잔액이 마이너스인 계좌는 빨간 테두리와 ⚠️ 경고가 표시됩니다. 기준잔액을 확인하세요.

📝 결재란 설정

보고서 상단 결재란에 표시될 항목 이름(예: 담당, 이사)을 변경합니다.

  • 1
    ⚙️ 설정 탭 → 결재란 설정 카드에서 항목 1, 항목 2 이름을 입력합니다.
  • 2
    ✅ 결재란 저장 버튼을 클릭합니다.
7 데이터 백업 & 복원

데이터는 PC에 자동 저장되지만, 정기적으로 백업해두면 PC 교체나 실수로 삭제했을 때 안전합니다.

💾 JSON 백업 (권장)

백업 파일(v2 형식)에는 거래내역 + 계좌설정 + 잔액 + 결재란 이름이 모두 포함됩니다.

  • 1
    메뉴 파일 → 데이터 백업 (JSON) 또는 Ctrl + S
    또는: 💾 내보내기 탭 → 데이터 관리 → 💾 JSON 백업
  • 2
    저장 위치를 선택하고 파일을 저장합니다.

📂 JSON 복원

  • 1
    메뉴 파일 → 백업 파일 복원 또는
    💾 내보내기 탭 → 데이터 관리 → 📂 JSON 복원
  • 2
    백업해두었던 .json 파일을 선택합니다.
⚠️ v2 형식(최신) 백업 복원 시 현재 데이터 전체가 백업 내용으로 덮어써집니다. 복원 전에 현재 데이터를 먼저 백업해두세요.
구버전 백업(거래내역만 있는 배열 형식)은 기존 내역과 병합됩니다.

🗑️ 전체 데이터 삭제

💾 내보내기 탭 → 데이터 관리 → 🗑️ 전체 삭제 버튼을 누르면 거래내역·계좌설정·잔액·결재란 이름이 모두 삭제됩니다. 2회 확인 후 삭제가 진행되며, 삭제 후에는 복구가 불가능합니다.

데이터 저장 위치 확인

메뉴 도움말 → 데이터 저장 위치 열기를 클릭하면 실제 데이터 파일이 저장된 폴더가 열립니다.

✅ 정기 백업 권장: 월 1회 이상 JSON 백업 파일을 USB 또는 클라우드에 보관하세요.

9 자주 묻는 질문 (FAQ)

Q. 잔액이 실제와 다르게 표시됩니다.

⚙️ 설정 탭에서 기준 잔액이 정확히 입력되어 있는지 확인하세요. 기준잔액을 기준으로 이후 입출금 내역이 더해지거나 빠지기 때문에, 기준값이 맞지 않으면 잔액이 틀립니다. 옆에 표시되는 현재잔액이 실제 잔액과 일치하는지 확인하세요.

Q. 기준잔액과 현재잔액의 차이가 무엇인가요?

기준 잔액은 처음 입력한 시작 금액입니다. 직접 수정 가능합니다.
현재잔액은 기준잔액 + 모든 입금 − 모든 출금으로 자동 계산된 값입니다. 수정할 수 없습니다.
가용자금 합계 화면의 잔액은 현재잔액을 기준으로 합산됩니다.

Q. 프로그램을 삭제하면 데이터도 사라지나요?

데이터는 프로그램과 별도 위치(사용자 앱 데이터 폴더)에 저장되므로 프로그램 삭제와 무관합니다. 단, 해당 폴더를 직접 삭제하거나 PC를 초기화하면 데이터가 손실되므로 정기적인 JSON 백업을 권장합니다.

Q. PC를 교체할 때 데이터를 옮기려면?

  • 1
    기존 PC: 파일 메뉴 → 데이터 백업(JSON)으로 백업 파일 저장
  • 2
    새 PC에 프로그램 설치
  • 3
    파일 메뉴 → 백업 파일 복원으로 백업 파일 불러오기 (계좌설정·잔액·거래내역 모두 복원됨)

Q. 전체 삭제 후 계좌가 하나도 없어졌습니다.

전체 삭제는 계좌설정까지 모두 삭제합니다. ⚙️ 설정 탭 → ➕ 계좌 추가로 계좌를 다시 추가하세요. 또는 이전에 저장한 JSON 백업 파일을 복원하면 계좌설정이 그대로 복구됩니다.

Q. 은행 파일 가져오기가 실패합니다.

  • 올바른 은행을 선택했는지 확인 (또는 자동 감지 시도)
  • 파일 형식이 .xlsx, .xls, .csv인지 확인
  • 파일이 Excel 등 다른 프로그램에서 열려 있지 않은지 확인 후 닫고 시도
  • 📋 은행별 엑셀 컬럼 구조 보기에서 해당 은행의 컬럼 구조를 확인

Q. 실수로 입력한 내역을 수정하려면?

현재 버전에서는 직접 수정 기능이 없습니다. 잘못된 내역을 삭제한 후 올바른 내용으로 다시 입력하세요.

Q. 마이너스 잔액이 표시됩니다.

출금 합계가 기준잔액 + 입금 합계를 초과하면 마이너스가 됩니다. 계좌 카드에 ⚠️ 잔액 부족 경고가 표시되고, 금액도 빨간 색으로 표시됩니다. 기준잔액이 올바른지 확인하거나, 누락된 입금 내역이 있는지 확인하세요.
