🏦 은행 거래내역 가져오기

은행 앱 또는 인터넷뱅킹에서 내려받은 거래내역 엑셀(.xlsx/.xls) 또는 CSV 파일을 불러옵니다.
기업은행 · 농협 · 우리은행 형식을 자동으로 인식합니다.

📌 은행 앱에서 내보내는 방법
기업은행: 거래내역 조회 → 엑셀 저장
농협: 거래내역 조회 → 내보내기 → 엑셀
우리은행: 거래내역 → 파일저장(Excel)
내보내는 방법도 확인할 수 있어요
