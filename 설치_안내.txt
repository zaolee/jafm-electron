■ 이 프로그램은 무엇인가요?
------------------------------------------------------------
소규모 기업용 자금 입출금 관리 프로그램입니다.
엑셀 없이도 간편하게 입출금 내역을 기록하고, 주간 보고서를
자동으로 만들어 드립니다. 인터넷 없이 내 PC에서만 동작하며,
모든 데이터는 내 컴퓨터에 안전하게 저장됩니다.


================================================================
  STEP 1.  설치 방법
================================================================

1. 설치 파일을 더블클릭합니다.
   파일명: 자금일보 Setup 1.0.0.exe

2. "이 앱이 디바이스를 변경하도록 허용하시겠어요?" 창이 뜨면
   → 예(Y) 를 클릭합니다.

3. 설치 마법사 화면이 열리면 Next(다음) 버튼을 클릭합니다.

4. 설치 경로를 확인하고 Install(설치) 버튼을 클릭합니다.
   (기본 경로: C:\Program Files\자금일보)

5. 설치가 완료되면 Finish 버튼을 클릭합니다.

6. 바탕화면 또는 시작 메뉴에서 "자금일보" 아이콘을 더블클릭하여
   실행합니다.

※ Windows 보안 경고가 뜨는 경우:
   "추가 정보" → "실행" 을 클릭하면 설치를 진행할 수 있습니다.


================================================================
  STEP 2.  처음 실행 시 설정
================================================================

프로그램을 처음 실행하면 아래 화면이 나타납니다.
상단 탭 중 ⚙️ 설정 탭을 먼저 설정해주세요.

1. 화면 상단의 "⚙️ 설정" 탭을 클릭합니다.

2. 각 계좌의 "기준 잔액" 칸에 현재 실제 잔액을 입력합니다.
   예) 기업은행 주계좌에 5천만원이 있다면 → 50000000 입력
       (콤마 없이 숫자만 입력하면 자동으로 콤마가 표시됩니다)

3. 계좌 이름이나 계좌번호가 다르면 수정하세요.
   사용하지 않는 계좌는 🗑️ 버튼으로 삭제할 수 있습니다.
   새 계좌를 추가하려면 "➕ 계좌 추가" 버튼을 클릭합니다.

4. 설정이 끝나면 반드시 "✅ 저장" 버튼을 클릭합니다.


================================================================
  STEP 3.  기본 사용 방법
================================================================

[입출금 내역 입력]
  1. "📋 자금 입력" 탭 클릭
  2. 날짜, 계좌, 입금/출금 구분, 금액, 거래처/내용 입력
  3. "✅ 저장" 클릭 → 하단 목록에 바로 추가됩니다

[보고서 생성 및 인쇄]
  1. "📊 보고서" 탭 클릭
  2. 조회할 기간 선택 (예: 월요일 ~ 일요일)
  3. "🔍 보고서 생성" 클릭
  4. "🖨️ 인쇄 미리보기" 클릭하여 인쇄 또는 PDF 저장
  5. "📄 Word 저장" 클릭하면 Word 파일로도 저장 가능

[엑셀 파일 저장]
  1. "💾 내보내기" 탭 클릭
  2. 기간 선택 후 "📥 엑셀 다운로드" 클릭

[은행 거래내역 가져오기]
  1. 은행 앱/인터넷뱅킹에서 거래내역을 엑셀 파일로 내보내기
  2. "💾 내보내기" 탭 → "🏦 은행 거래내역 가져오기" 섹션
  3. 은행 선택 → 계좌 선택 → "📂 파일 선택" 클릭
  4. 미리보기 확인 후 "✅ 선택 항목 가져오기" 클릭


================================================================
  STEP 4.  데이터 백업 (꼭 해주세요!)
================================================================

프로그램을 오래 사용할수록 데이터가 중요해집니다.
정기적으로 백업해두면 PC 교체나 문제 발생 시 복구할 수 있습니다.

[백업 방법]
  방법 1: 키보드에서 Ctrl + S 키를 동시에 누릅니다.
  방법 2: 상단 메뉴 → 파일 → 데이터 백업 (JSON) 클릭
  방법 3: "💾 내보내기" 탭 → 데이터 관리 → "💾 JSON 백업" 클릭

저장 위치를 선택하면 날짜가 포함된 파일이 저장됩니다.
예) 자금일보_백업_2026-03-18.json

[복원 방법]
  방법 1: 상단 메뉴 → 파일 → 백업 파일 복원 클릭
  방법 2: "💾 내보내기" 탭 → 데이터 관리 → "📂 JSON 복원" 클릭
  → 백업 파일을 선택하면 거래내역 + 계좌설정이 모두 복원됩니다.

★ 권장: 매월 1회 이상 백업 파일을 USB 또는 클라우드(네이버MYBOX,
         구글드라이브 등)에 보관하세요.


================================================================
  STEP 5.  데이터 저장 위치
================================================================

모든 데이터는 아래 폴더에 자동 저장됩니다.

  C:\Users\[사용자명]\AppData\Roaming\자금일보\

이 폴더를 직접 열려면:
  상단 메뉴 → 도움말 → 데이터 저장 위치 열기 클릭

주의: 이 폴더를 삭제하면 데이터가 모두 사라집니다.
      반드시 JSON 백업을 먼저 해두세요.


================================================================
  자주 묻는 질문
================================================================

Q. 윈도우 보안 경고가 떠서 설치가 안 됩니다.
A. "추가 정보" 링크를 클릭한 후 "실행" 버튼을 누르세요.

Q. 잔액이 실제랑 다릅니다.
A. ⚙️ 설정 탭에서 각 계좌의 "기준 잔액"을 확인하고
   올바른 금액으로 수정 후 저장하세요.

Q. 잔액이 빨간색으로 표시됩니다.
A. 출금 합계가 입금보다 많아 마이너스 상태입니다.
   기준잔액 또는 누락된 입금 내역을 확인하세요.

Q. 프로그램을 지워도 데이터가 남아있나요?
A. 네, 데이터는 별도 폴더에 저장되므로 프로그램 삭제와
   무관하게 보존됩니다. (단, 백업은 필수!)

Q. 사용 방법이 더 궁금합니다.
A. 프로그램 상단 오른쪽 "❓ 도움말" 버튼을 클릭하면
   상세 사용설명서(manual.html)가 열립니다.


================================================================
  프로그램 제거 방법
================================================================

Windows 설정 → 앱 → 앱 및 기능 → "자금일보" → 제거
또는 시작 메뉴 → 자금일보 → 마우스 오른쪽 클릭 → 제거

※ 제거해도 데이터 폴더는 삭제되지 않습니다.
   데이터도 함께 삭제하려면 위 "STEP 5"의 폴더를 직접 삭제하세요.
